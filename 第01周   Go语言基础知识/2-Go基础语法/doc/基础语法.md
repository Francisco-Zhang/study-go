## 定义变量

### 变量默认值
``` 
func variableZeroValue() {
	var a int
	var b string
	fmt.Println(a, b)
	fmt.Printf("%d %q\n", a, b) //打印空串标记
}
func main() {
	fmt.Println("Hello World")
	variableZeroValue()
}

输出
Hello World
0 
0 ""
```
### 变量赋初始值
```
func variableInitialValue() {
	var a, b int = 3, 4
	var s string = "abc"
	fmt.Println(a, b, s)
}

输出 3 4 abc
```
